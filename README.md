# ConfSpace

Демонстраційний веб-додаток для кваліфікаційної бакалаврської роботи на тему:

**Розробка веб-додатка для перегляду розкладу та матеріалів наукових конференцій**

## Як запустити у VS Code

1. Відкрити папку `conference-app` у VS Code.
2. Відкрити файл `index.html`.
3. Запустити через розширення Live Server або відкрити `index.html` у браузері.

Додаток не потребує встановлення npm-пакетів і працює як статичний сайт.

## Реалізовано

- перегляд списку конференцій;
- гіперпосилання для приєднання до online-конференцій;
- просте логінування з ролями користувачів;
- профіль користувача після входу;
- перегляд розкладу доповідей;
- обмеження доступу до розкладу та матеріалів до входу;
- відображення ролі спікера: студент, викладач, експерт тощо;
- статуси доповідей: заплановано, триває, завершено;
- пошук за назвою, секцією, спікером або описом;
- фільтрація за конференцією, секцією та роллю;
- друк розкладу;
- перегляд деталей конференції;
- перегляд матеріалів доповідей;
- відкриття внутрішнього вікна з матеріалами конкретної доповіді;
- адміністративна форма додавання нової доповіді;
- dashboard адміністратора з кількістю конференцій, секцій, доповідей і матеріалів;
- вибір режиму в адмін-панелі: створення доповіді або створення конференції;
- редагування доповідей, секцій, дати та часу через адмін-панель;
- видалення доповідей з розкладу;
- збереження доданих записів у `localStorage`;
- адаптивний інтерфейс для комп'ютера та смартфона.

## Демо-логіни

- `student@example.com` / реєструється через форму користувача
- `admin` / `1234` - адміністратор, може переглядати розклад, матеріали та додавати доповіді.

## Структура

- `index.html` - структура сторінки;
- `styles.css` - стилі інтерфейсу;
- `app.js` - логіка, демо-дані, пошук, фільтрація та робота з формою.
